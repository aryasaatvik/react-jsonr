---

React JSONR is available as an npm package, and you can install it using npm, yarn, or pnpm.

## Requirements

- React 18.0.0 or higher
- React DOM 18.0.0 or higher

## Installation

```bash
# Using npm
npm install react-jsonr

# Using yarn
yarn add react-jsonr

# Using pnpm
pnpm add react-jsonr
```

## What's Next?

After installation, you can start using React JSONR in your React project. Check out the [Basic Usage](/docs/getting-started/basic-usage) guide to learn how to render UI components from JSON.
